# 🎮 ワンナイト人狼オリジナル版 - 超初心者向けセットアップガイド

このガイドでは、プログラミング経験がない方でもゲームを起動できるよう、**ステップバイステップ**で説明します!

---

## 📥 ステップ1: ファイルをダウンロード

1. **ZIPファイルをダウンロード**
   - ダウンロードした `werewolf-project.zip` を確認してください
   - 通常、「ダウンロード」フォルダにあります

2. **ZIPファイルを解凍**
   - Windowsの場合:
     - ZIPファイルを**右クリック** → 「すべて展開」
     - 展開先を選択 (デスクトップがおすすめ)
     - 「展開」ボタンをクリック
   
   - Macの場合:
     - ZIPファイルを**ダブルクリック**
     - 自動的に解凍されます

3. **フォルダを確認**
   - `werewolf-project` というフォルダができます
   - 中を開くと、`server` と `client` フォルダがあります

---

## 💻 ステップ2: Visual Studio Codeで開く

### 方法1: VS Codeから開く

1. **VS Codeを起動**
   - デスクトップまたはスタートメニューから起動

2. **フォルダを開く**
   - メニューバー → 「ファイル」 → 「フォルダーを開く...」
   - または: **Ctrl + K → Ctrl + O** (Macは **Cmd + K → Cmd + O**)

3. **werewolf-projectを選択**
   - 先ほど解凍した `werewolf-project` フォルダを選択
   - 「フォルダーの選択」ボタンをクリック

### 方法2: エクスプローラー/Finderから開く

1. **werewolf-projectフォルダを右クリック**
   - 「Codeで開く」を選択
   - (この項目がない場合は方法1を使ってください)

---

## 📦 ステップ3: 必要なパッケージをインストール

ゲームを動かすために必要なパッケージをインストールします。

### 3-1: ターミナルを開く

VS Code内で:
1. **メニューバー** → 「ターミナル」 → 「新しいターミナル」
2. または: **Ctrl + `** (バッククォート) (Macは **Cmd + `**)

画面下部に「ターミナル」が表示されます。

### 3-2: サーバー側のパッケージをインストール

ターミナルに以下を**1行ずつ**入力して、Enterキーを押してください:

```bash
cd server
```
↑ これでserverフォルダに移動します

```bash
npm install
```
↑ これで必要なパッケージをインストールします

**待ちます (1〜2分かかります)**
- 画面に色々な文字が流れますが、正常です!
- エラーが出なければOK!

完了したら:
```bash
cd ..
```
↑ これで元のフォルダに戻ります

### 3-3: クライアント側のパッケージをインストール

続けて、ターミナルに入力:

```bash
cd client
```

```bash
npm install
```

**待ちます (1〜2分かかります)**
- またたくさんの文字が流れます
- エラーが出なければOK!

完了したら:
```bash
cd ..
```

### ✅ インストール完了!

これで必要なパッケージのインストールが完了しました!

---

## 🚀 ステップ4: ゲームを起動する

ゲームを起動するには、**サーバー**と**クライアント**の2つを同時に起動する必要があります。

### 4-1: ターミナルを2つに分割

現在のターミナルを分割します:

1. **方法A: ボタンで分割**
   - ターミナルの右上にある「➕」ボタンの横の「⊞」ボタンをクリック
   - 「ターミナルの分割」を選択

2. **方法B: 新しいターミナルを追加**
   - メニューバー → 「ターミナル」 → 「新しいターミナル」
   - もう1つターミナルが開きます

### 4-2: サーバーを起動 (左側のターミナル)

**左側のターミナル**に入力:

```bash
cd server
```

```bash
npm start
```

成功すると、以下のように表示されます:
```
🎮 ワンナイト人狼サーバー起動: http://localhost:3001
```

**このターミナルはそのままにしておいてください!**

### 4-3: クライアントを起動 (右側のターミナル)

**右側のターミナル**に入力:

```bash
cd client
```

```bash
npm start
```

しばらく待つと...

✨ **自動的にブラウザが開きます!** ✨

- URL: `http://localhost:3000`
- ゲーム画面が表示されます!

### ❗ ブラウザが開かない場合

手動で開いてください:
1. ブラウザ (Chrome, Edge, Firefoxなど) を開く
2. アドレスバーに `http://localhost:3000` と入力
3. Enterキーを押す

---

## 🎮 ステップ5: 友達と遊ぶ

### 5-1: ルームを作成

1. **名前を入力**
   - 「あなたの名前を入力」に名前を入力

2. **ルームを作成**
   - 「新しいルームを作成」ボタンをクリック
   - ルームIDが表示されます (例: ROOM-AB12CD)

3. **ルームIDをメモ**
   - このIDを友達に教えます!

### 5-2: 友達が参加する

友達にやってもらうこと:

1. **同じURL にアクセス**
   - ブラウザで `http://あなたのIPアドレス:3000` を開く
   - (ローカルネットワークの場合)
   - または、同じPCで別のブラウザタブを開く

2. **名前を入力**

3. **ルームIDを入力**
   - あなたから教えてもらったルームIDを入力
   - 「ルームに参加」をクリック

### 5-3: 役職を設定してゲーム開始

1. **ホスト(あなた)が役職を設定**
   - 「役職を設定する」ボタンをクリック
   - +/- ボタンで各役職の枚数を調整
   - **必ず「プレイヤー数 + 3枚」になるようにしてください**

2. **ゲーム開始**
   - 「ゲーム開始」ボタンをクリック

3. **遊ぶ!**
   - 各自の画面に役職が表示されます
   - Discord や Zoom で通話しながら遊ぶと楽しいです!

---

## 🛑 ゲームを終了する方法

### ゲームを止める時

1. **両方のターミナルで `Ctrl + C` を押す**
   - サーバー側のターミナル: Ctrl + C
   - クライアント側のターミナル: Ctrl + C

2. **ターミナルを閉じる**

3. **VS Codeを閉じる**

### 次回起動する時

**ステップ4**からやり直せばOKです!
(パッケージのインストールは1回だけでOK)

## ❓ よくある質問

### Q: 「npm: コマンドが見つかりません」と表示される

**A:** Node.jsがインストールされていません。

1. https://nodejs.org/ にアクセス
2. 「推奨版」をダウンロード
3. インストール後、PCを再起動
4. もう一度ステップ3からやり直す

### Q: ポート3000が使用中と表示される

**A:** 他のアプリが3000番ポートを使っています。

**解決方法1:** 他のアプリを閉じる
**解決方法2:** 別のポートで起動

```bash
# クライアントを3002番ポートで起動
PORT=3002 npm start
```

その後、`http://localhost:3002` にアクセス

### Q: 友達が参加できない

**A:** 以下を確認してください:

1. **同じWi-Fiに接続しているか?**
   - 同じネットワークにいる必要があります

2. **IPアドレスを確認**
   - Windowsの場合: コマンドプロンプトで `ipconfig`
   - Macの場合: ターミナルで `ifconfig`
   - 友達には `http://あなたのIPアドレス:3000` を教える

3. **ファイアウォールを確認**
   - ファイアウォールでポート3000, 3001が開いているか確認

### Q: エラーが出た!

**A:** 以下を試してください:

1. **node_modulesを削除して再インストール**
   ```bash
   # serverフォルダで
   rm -rf node_modules
   npm install
   
   # clientフォルダで
   rm -rf node_modules
   npm install
   ```

2. **VS Codeを再起動**

3. **PCを再起動**
