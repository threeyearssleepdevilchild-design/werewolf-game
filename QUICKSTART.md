# 🚀 クイックスタートガイド

**5分でゲームを起動する方法!**

---

## 📥 1. ダウンロードと解凍

1. `werewolf-project.zip` をダウンロード
2. ZIPファイルを**右クリック** → 「すべて展開」(Windows) または**ダブルクリック**(Mac)
3. デスクトップに展開

---

## 💻 2. VS Codeで開く

1. **Visual Studio Code** を起動
2. **ファイル** → **フォルダーを開く**
3. `werewolf-project` フォルダを選択

---

## 📦 3. パッケージをインストール (初回のみ)

### ターミナルを開く
- **Ctrl + `** (バッククォート) を押す

### サーバー側
```bash
cd server
npm install
cd ..
```

### クライアント側
```bash
cd client
npm install
cd ..
```

**※ 1〜2分待つ**

---

## 🎮 4. ゲームを起動

### ターミナルを2つに分割
- ターミナルの右上「⊞」ボタンをクリック

### 左側のターミナル: サーバー起動
```bash
cd server
npm start
```

### 右側のターミナル: クライアント起動
```bash
cd client
npm start
```

**✨ 自動的にブラウザが開きます!**

---

## 🎉 5. 遊ぶ!

1. **名前を入力**
2. **「新しいルームを作成」** をクリック
3. **ルームIDを友達に教える**
4. **友達が参加したら役職を設定**
5. **ゲーム開始!**

---

## 📞 友達と遊ぶ方法

### 友達にやってもらうこと:
1. ブラウザで `http://localhost:3000` を開く (同じPC)
2. または `http://あなたのIPアドレス:3000` を開く (別のPC)
3. 名前を入力
4. ルームIDを入力して参加

**推奨:** Discord/Zoomで通話しながら遊ぶと楽しい!

---

## 🛑 終了方法

**両方のターミナルで `Ctrl + C` を押す**

---

## ❓ トラブル?

### コマンドが見つからない
→ Node.jsをインストール: https://nodejs.org/

### ポート3000が使用中
→ `PORT=3002 npm start` で起動

### 友達が参加できない
→ 同じWi-Fiに接続しているか確認

---
