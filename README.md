## 🚀 セットアップ手順

### 1. 必要なソフトウェア

以下がインストール済みであることを確認してください:
- Node.js (v14以上)
- npm (Node.jsに含まれています)

### 2. プロジェクトのセットアップ

#### Windows の場合

1. **コマンドプロンプトを開く**
   - Windowsキー + R → `cmd` と入力 → Enter

2. **プロジェクトフォルダに移動**
   ```cmd
   cd C:\Users\あなたのユーザー名\Downloads\werewolf-project
   ```

3. **サーバーの依存関係をインストール**
   ```cmd
   cd server
   npm install
   cd ..
   ```

4. **クライアントの依存関係をインストール**
   ```cmd
   cd client
   npm install
   cd ..
   ```

#### Mac / Linux の場合

1. **ターミナルを開く**
   - Command + Space → `terminal` と入力

2. **プロジェクトフォルダに移動**
   ```bash
   cd ~/Downloads/werewolf-project
   ```

3. **サーバーの依存関係をインストール**
   ```bash
   cd server
   npm install
   cd ..
   ```

4. **クライアントの依存関係をインストール**
   ```bash
   cd client
   npm install
   cd ..
   ```

## 🎮 ゲームの起動方法

### 方法1: 2つのターミナルを使う (推奨)

#### ターミナル1: サーバー起動

```bash
cd server
npm start
```

成功すると以下のように表示されます:
```
🎮 ワンナイト人狼サーバー起動: http://localhost:3001
```

#### ターミナル2: クライアント起動

```bash
cd client
npm start
```

自動的にブラウザが開き、`http://localhost:3000` が表示されます。

### 方法2: VS Codeを使う

1. **VS Codeでプロジェクトフォルダを開く**
   - ファイル → フォルダを開く → `werewolf-project` を選択

2. **ターミナルを2つ開く**
   - ターミナル → ターミナルを分割 (または新しいターミナル)

3. **1つ目のターミナル: サーバー起動**
   ```bash
   cd server
   npm start
   ```

4. **2つ目のターミナル: クライアント起動**
   ```bash
   cd client
   npm start
   ```

## 🎯 遊び方

### 1. ゲームを始める

1. ブラウザで `http://localhost:3000` を開く
2. 名前を入力
3. 「新しいルームを作成」をクリック
4. 表示されたルームIDを友達に教える

### 2. 友達が参加する

1. 友達もブラウザで `http://localhost:3000` を開く
2. 名前を入力
3. ルームIDを入力して「ルームに参加」

### 3. ゲーム開始

1. ホスト(最初に作った人)が「役職を設定する」をクリック
2. 役職の枚数を調整 (プレイヤー数 + 3枚)
3. 「ゲーム開始」をクリック

### 4. 夜フェーズ

- 各プレイヤーに役職が配られます
- 役職に応じて能力を使います

### 5. 昼フェーズ

- 議論時間 (3〜5分)
- 自分の情報を共有したり、嘘をついたりして推理します
- Discord や Zoom などで通話しながら遊ぶのがおすすめ!

### 6. 投票フェーズ

- 処刑したいプレイヤーを選んで投票

### 7. 結果発表

- 勝敗が表示されます!

## ⚠️ トラブルシューティング

### ポート3000が使用中の場合

```bash
# クライアントを別のポートで起動
PORT=3002 npm start
```

### サーバーに接続できない

1. サーバーが起動しているか確認
2. `http://localhost:3001` にアクセスして確認
3. ファイアウォールを確認

### モジュールが見つからない

```bash
# 依存関係を再インストール
npm install
```

## 🔧 開発者向け情報

### プロジェクト構成

```
werewolf-project/
├── server/              # バックエンド (Node.js + Socket.io)
│   ├── package.json
│   └── server.js       # サーバーメインファイル
└── client/             # フロントエンド (React)
    ├── package.json
    ├── public/
    └── src/
        ├── App.js      # メインコンポーネント
        ├── socket.js   # Socket.io接続
        └── components/ # 各画面コンポーネント
```

- **フロントエンド**: React 18
- **バックエンド**: Node.js + Express
- **リアルタイム通信**: Socket.io
- **状態管理**: React Hooks
